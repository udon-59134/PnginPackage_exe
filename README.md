# PnginPackage_exe
コンパイラ済みのexeでの配布  
## 初めてのgithubへのアップロード
こちら( https://twitter.com/David3141593/status/1372083857219059714 )で紹介されていた技術とソースコードを利用して、一般の方にも使えるようにC#でGUIを作成しました。  
pyinstallerを用いて、exe化したものをラッピングして外部呼び出しを行ってるだけのものとなります。
